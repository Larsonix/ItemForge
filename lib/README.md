# Dependencies

Place external dependency JARs here. These are bundled into the output JAR
via the Shadow plugin.

## Vuetale.jar

**Required.** Download from:
- [GitHub Releases](https://github.com/KelpyCode/Vuetale/releases)
- [Modtale](https://modtale.net/mod/vuetale)
- [CurseForge](https://www.curseforge.com/hytale/mods/vuetale)

Rename the downloaded file to `Vuetale.jar` and place it here.

Vuetale is bundled into ItemForge's JAR — server owners install one plugin
and get everything. They never need to install Vuetale separately.
